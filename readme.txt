Wordpress Mailchimp Subscription Custom Form

== Description ==

"Mailchimp Subscription Custom Form" plugin is a mailchimp subscription popup.
its gives custom popup and functionality to enable mailchimp subscription for your wordpress site. you can design your popup from admin options and can give captions from admin.

You will be able to:

	
= KEY BENEFITS: =
- create custom popup for mailchimp subscription. 
- you can give different colors in popup.
- you can define mailchimp subscription fields.
- you can define border radious for text box and popup.
- you can design submit button and its color from admin panel.
- rich color picker are used in admin to design mailchimp subscription popup.

== Installation ==

1) Copy/Upload 'gen-mailchimp-wordpress-plugin' folder to the '/wp-content/plugins/' directory
2) Activate 'Mailchimp Subscription Custom Form' from wp plugin option in admin area
3) Plugin will appear in the menu bar of WP Dashboard
4) set your mailchimp api key and list id in admin -> mailchimp settings plugin settings page.
5) Design and setup your popup from admin->settings for popup
